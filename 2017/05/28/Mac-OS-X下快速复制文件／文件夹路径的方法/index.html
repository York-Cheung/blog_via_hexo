Mac OS X下快速复制文件／文件夹路径的方法

发表于 2017-05-28 | 分类于 教程 |

做开发过程中有时候会需要在代码中使用文件路径。用习惯了Windows直接在路径栏复制文件路径的方式，Mac用户会发现OSX下没有一个方便的方法复制文件或文件夹的路径，笔者都是用鼠标右键文件然后在显示简介里复制文件路径，但还需要手动添加上文件名，十分的不人性化💩💩💩，经过一番google，发现了一个用workflow添加右键快捷菜单项的方法，故分享在此。

在Launchpad里打开Automator：

在左侧操作栏找到拷贝至剪贴板操作，拖到右侧，如下

有两个地方需要修改，"服务"收到选定的选择“文件或文件夹”，位于选择Finder。

最后command+s保存，命名为copy path，Done！

接下来试一下效果，随便找一个文件，右键一下，看看菜单里是不是多了一个copy path选项。文件夹的快捷菜单项在服务里。点击之后文件路径就被复制到了剪贴板，找个地方粘贴一下看看结果是不是你想要的，如果想删除该服务，进入~/Library/Services/目录，删除对应的workflow文件即可。

至此，快速复制文件路径的配置已经完成，默默吐槽一下苹果爸爸不把这一功能添加到系统里。
